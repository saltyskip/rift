# Demo Themes

These sample payloads demonstrate the landing theme system with three distinct brands.

- The JSON files use placeholder CDN-style URLs for API examples.
- Matching checked-in preview assets live under `marketing/public/demo-themes/<slug>/`.
- If you want to demo these through the API locally, replace the asset URLs with your own hosted equivalents.

Assets available locally:

- `marketing/public/demo-themes/nord-roast/`
- `marketing/public/demo-themes/volt-run/`
- `marketing/public/demo-themes/atelier-stay/`
